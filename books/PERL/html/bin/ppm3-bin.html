NAME

ppm3-bin - ppm3 executable

SYNOPSIS

Do not run ppm3-bin manually. It is meant to be called by the wrapper program ppm3. See ppm3.

DESCRIPTION

ppm3 runs ppm3-bin after setting up a few environment variables. You should run ppm3 instead.

For information about ppm3 commands, see ppm3.
